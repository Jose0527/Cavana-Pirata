La transmisión es una técnica poderosa en la que los datos se transmiten desde un origen a un destino.

Con la transmisión, la forma de crear un cálculo y luego leerlo todo junto es usar un sistema de editor-suscriptor. El editor es el sistema de archivos, generalmente el sistema de archivos CMD.

El suscriptor suele ser el kernel de Linux. El núcleo se ejecuta como suscriptor y, en el caso de http, puede ver cuántos suscriptores se están ejecutando actualmente.
